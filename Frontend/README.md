# Nepfluence

Nepfluence is a marketplace for Nepali and Indian brands to discover creators, run influencer campaigns, manage collaborations, and protect payouts through an escrow-first workflow.

## Frontend

This app uses Next.js App Router, TypeScript, TailwindCSS, shadcn-style UI primitives, and domain-oriented feature folders. See `ARCHITECTURE.md` for the project structure and boundaries.

## Scripts

- `npm run dev` starts the local Next.js server.
- `npm run build` creates a production build.
- `npm run lint` runs ESLint.
